# CheatSheet


## Flask

Display blueprints/routes
```bash
flask routes
```

Access the terminal
```bash
flask shell
```

---

## Python

Use `help()` to display help docs of an object
```python
from app.models.user import User
help(User)
```

Use `dir()` to list the methods and attributes of an object
```python
from app.models.user import User
dir(User)
```
